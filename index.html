Vim is a language for editing text. Once you can speak that language, editing just happens naturally.

Basic movement

  • w: move forward one 'word'
  • b: move back one 'word'
  • j: move down one line
  • k: move up one line
  • h: move left one character
  • l: move right one character

Insert

  • a: after caret
  • i: before caret
  • A: after line
  • I: before line

Delete

  • d: delete
  • D: delete to the end of the line
  • dd: delete the whole line
  • x: delete the character at caret
  • X: delete the character before caret

    • Search

  • f: find next occurance of a character on the line
  • F: find previous occurance of a character on the line
  • /: search for typed characters in the document
    • Vim also has a concept of special actions called motions. Motions are combinations of single commands that work together to perform some action like deleting a word Many of these motions are mnemonic and easy to remember.
