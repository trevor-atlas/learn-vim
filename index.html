So what is Vim?

💾 Open Source Text editor created in 1991, based on an earlier text editor (Vi) created in 1976 for UNIX

🖥️ Terminal based, but can be hosted in a GUI (more on this later)

🌎 All *NIX based OS ship with some variation of Vim or Vi

Vim is a language for editing text Once you can speak that language, editing just happens naturally. insert mode w: move forward one 'word' b: move back one 'word' f: find next occurance of a character on the line F: find previous occurance of a character on the line a: after caret i: before caret A: after line I: before line d: delete D: delete to the end of the line dd: delete the whole line
motions the main power of vim is in motions. motions are combinations of single commands that work together to perform some action like deleting a word
